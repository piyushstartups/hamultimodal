**Key Technical Concepts**
-   **Client-side Image Compression:** Using browser APIs to resize and compress images before uploading to save bandwidth and time.
-   **UI State Lifting:** Managing state (like which item is expanded) in a parent component to control multiple child components and create accordion-style behavior.
-   **Lazy Loading:** Deferring the loading of data (especially images) until it is needed by the user to improve initial page load performance.
-   **Backend-Enforced Access Control (RBAC):** Implementing authorization checks directly in the API endpoints to ensure users can only perform actions they are permitted to, regardless of the frontend UI.
-   **Database-level Concurrency Control:** Preventing duplicate active records for the same resource (e.g., a kit in a shift) to ensure data integrity across multiple devices.
